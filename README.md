# dotfiles

dotfiles!

Written for macOS and Ubuntu.

## Tools
- OMZ + p10k
- fzf
- ripgrep
- bat
- eza
- zoxide
- delta
- tldr

## Installation

```
sudo apt update && sudo apt install -y git
git clone --recurse-submodules https://github.com/Andrie-Amor/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

Update the nvim submodule

```
git submodule update --remote
```
